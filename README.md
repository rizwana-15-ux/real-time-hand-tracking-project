Real-Time Hand Tracking Project

This is an independent, Python-based computer vision project that tracks human hands in real-time. 

Features
* Real-Time Detection: Uses webcam input to detect hands instantly.
* High Accuracy: Built using OpenCV and MediaPipe framework for precise landmark tracking.
* Independent Implementation:** Fully structured and developed from scratch.

 How to Run
1. Extract the uploaded `.zip` folder.
2. Install the required dependencies (OpenCV, MediaPipe).
3. Run the main execution script.
