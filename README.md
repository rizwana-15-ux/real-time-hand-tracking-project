## ✨ Real-Time Hand Tracking Project: Neon Aura AR

An immersive, browser-based **Real-Time Hand Tracking** experience that transforms hand movements into **interactive neon visuals and responsive audio effects** using a live webcam feed.

Built entirely with **web technologies**, the project runs instantly in modern browsers — without requiring Python setup, heavy installations, or specialized hardware.

## 🚀 Project Overview

Neon Aura AR captures live webcam input and converts physical hand gestures into real-time digital interactions using **computer vision, dynamic visuals, and audio synthesis**, creating an immersive AR-like browser experience.

---

## 🖐️ Real-Time Hand Tracking (MediaPipe)

Using **MediaPipe Hands**, the system performs browser-based hand tracking in real time.

### ✨ Features
- **21 Landmark Mapping** – Tracks fingertips, joints, and knuckles with precision.
- **Dual Hand Detection** – Supports tracking of **up to two hands simultaneously**.
- **Smooth Performance** – Uses browser hardware acceleration for responsive tracking.
- **Client-Side Processing** – Runs fully in-browser with **no backend dependency**.

---

## 🎨 Neon Visual Effects (HTML5 Canvas)

The application generates futuristic neon visuals using layered **HTML5 Canvas rendering**.

### ✨ Visual Interactions
- **Fingertip Particles** – Creates glowing spark trails from fingertip movement.
- **⚡ Cross-Hand Lightning** – Generates electric arcs when hands come close.
- **🌌 Dynamic Background** – Matrix-style digital rain reacts to movement speed.
- **🎭 Theme Switching** – Choose between **Cyberpunk, Lava, and Galaxy** themes.

---

## 🎵 Gesture Recognition & Audio Interaction

The system interprets gestures and produces responsive sound effects using the **Web Audio API**.

### ✨ Interactive Features
- **✋ Posture Detection** – Recognizes gestures like **Open Hand** and **Fist**.
- **🤏 Pinch Gesture** – Triggers a **“PINCH!”** event with electric sound effects.
- **🔊 Distance-Based Audio** – Hand proximity controls **pitch and volume dynamically**.

---

## 🛠️ Tech Stack

- **JavaScript (ES6+)** – Logic & interactions  
- **MediaPipe Hands** – Hand tracking  
- **HTML5 Canvas** – Visual rendering  
- **Web Audio API** – Interactive audio synthesis  
- **CSS3** – Styling & animations
